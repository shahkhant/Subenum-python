# Subdomain Enumeration Tool

A fast, Python-based subdomain enumeration tool that performs DNS brute-forcing with multi-threading and optional HTTP/HTTPS probing. Designed for reconnaissance, penetration testing, and security research.

---

## Features
- DNS-based subdomain enumeration
- Multi-threaded for faster scanning
- Optional HTTP/HTTPS probing
- Simple command-line interface
- Lightweight and easy to use

---

## Requirements
- Python 3.10 or higher
- Internet connection

---

## Installation

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/subenum-python.git
cd subenum-python
```

---

### 2. CODE
```bash
python subdomain.py -d example.com -w wordlist.txt
```
